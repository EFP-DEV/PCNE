# CHECKLIST PARTICIPANTS – Cybersécurité & Ergonomie

## 1. Vigilance physique

### Supports amovibles (clés USB, câbles, chargeurs)

| Situation courante                      | Agir (solution)                                          | Backup utile ? |
| --------------------------------------- | -------------------------------------------------------- | -------------- |
| Clé USB trouvée ou donnée par quelqu’un | Ne jamais brancher, utiliser uniquement ses propres clés | Non            |
| Chargeur ou câble prêté                 | Utiliser uniquement un chargeur/câble personnel certifié | Non            |
| Disque dur externe                      | Utiliser un disque chiffré (BitLocker, VeraCrypt)        | Oui ☐          |

### Vol ou perte d’appareil

| Appareil            | Agir (solution)                                | Backup utile ? |
| ------------------- | ---------------------------------------------- | -------------- |
| Ordinateur portable | Activer chiffrement disque + housse antivol    | Oui ☐          |
| Smartphone          | Code PIN/biométrie + localisation activée      | Oui ☐          |
| Disque dur externe  | Stockage sécurisé (coffre/tiroir fermé)        | Oui ☐          |
| Clé USB personnelle | Clé sécurisée avec mot de passe ou chiffrement | Oui ☐          |

### Postes laissés sans surveillance

| Situation courante   | Agir (solution)                     |
| -------------------- | ----------------------------------- |
| Bureau quitté 2 min  | Verrouillage auto (30 sec – 1 min)  |
| Pause-café / réunion | Bureau fermé à clé                  |
| Fin de journée       | Câble antivol ou rangement sécurisé |

### Connexions réseau

| Situation courante         | Agir (solution)                 |
| -------------------------- | ------------------------------- |
| Wi-Fi public (café, hôtel) | Utiliser un VPN fiable          |
| Partage de connexion       | Configurer partage sécurisé     |
| VPN professionnel          | Utiliser un fournisseur reconnu |

### Risques de destruction matérielle

| Menace possible            | Agir (solution)                          | Backup utile ? |
| -------------------------- | ---------------------------------------- | -------------- |
| Surtension (foudre, panne) | Multiprise avec parasurtenseur           | Oui ☐          |
| Incendie / dégât des eaux  | Ranger supports hors du bureau           | Oui ☐          |
| Maladresse humaine         | Housse de protection, boîtiers sécurisés | Oui ☐          |

## 3. Détection & réaction

### Cas typiques

| Situation courante           | Agir (solution)                   | Qui prévenir ? |
| ---------------------------- | --------------------------------- | -------------- |
| Mail suspect                 | Vérifier adresse, ne pas cliquer  | …              |
| Alerte antivirus             | Mettre à jour, contacter support  | …              |
| Clé USB inconnue             | Ne jamais brancher                | …              |

### Réflexes universels

* Ne pas cliquer immédiatement
* Demander avis à un collègue
* Isoler l’appareil du réseau

## 4. Smartphone

### Sécurité de base

| Paramètre    | Agir (solution)              | Risque si absent             |
| ------------ | ---------------------------- | ---------------------------- |
| Code PIN     | Code robuste 6 chiffres min. | Accès complet au téléphone   |
| Biométrie    | Activer empreinte/face ID    | Usurpation par code faible   |
| Mises à jour | Automatiques activées        | Exploitation failles connues |

### Sauvegarde

| Action            | Agir (solution)                | Risque si absent        |
| ----------------- | ------------------------------ | ----------------------- |
| Photos/messages   | Sauvegarde automatique activée | Perte définitive        |
| Test restauration | Essai sur un autre appareil    | Sauvegarde inutilisable |

### Authentification forte

| Compte sensible | Agir (solution)                       | Risque si absent  |
| --------------- | ------------------------------------- | ----------------- |
| Mail, banque…   | Activer double authentification (2FA) | Piratage possible |

### Applications

| Action               | Agir (solution)                     | Risque si absent       |
| -------------------- | ----------------------------------- | ---------------------- |
| Supprimer inutiles   | Nettoyage régulier                  | Failles dormantes      |
